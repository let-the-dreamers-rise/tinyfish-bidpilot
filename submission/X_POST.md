## Follow-Up Thread

1. Proposal teams do not just need better drafting. They need help inside live procurement portals.
2. BidPilot focuses on one workflow: save draft, stop at review.
3. TinyFish is the core runtime handling live browser execution, sessions, and workflow state.
4. We added guardrails for `read-only` and `draft-save` policies so the agent stays useful without becoming reckless.
5. The biggest insight: the real value is in boring portal labor, not another chat wrapper.
